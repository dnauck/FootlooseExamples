Public Fields
  Name Description
  Error A message that reports an error that has occurred regarding processing or delivery of a previously-sent get or set request.

A communication endpoint hat receives a message of type Result must not response with any further message of type Result or Error.

  Get A message that requests information, inquires about what data is needed in order to complete further operations, etc.

A communication endpoint hat receives a message of type Get must reply with a message of type Result or Error.

The response must preserve the 'id' attribute of the request.

  Notification A message that notify other endpoints about certain events.

This is a one-way message. A communication endpoint hat receives a message of type Notification must not response with any further message of type Result or Error.

  Result A message that is a response to a successful get or set request.

A Result message must not include any message body.

A communication endpoint hat receives a message of type Result must not response with any further message of type Result or Error.

  Set A message that provides data.

A communication endpoint hat receives a message of type Set must reply with a message of type Result or Error.

The response must preserve the 'id' attribute of the request.
