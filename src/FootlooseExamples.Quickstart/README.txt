Footloose Quickstart Example

1.) Description

This example shows how to configure and create a very simple client and service
using the .NET remoting IPC transport channel with the Footloose fluent configuration API.



2.) How to build?

Use the provided "build-all.bat" in the Examples root directory or open the
"FootlooseExamples.Quickstart.sln" solution file with VS2010 and run
Build -> Build Solution in the VS2010 menu.



3.) How to run?

You'll find the binaries of the client in "Build\FootlooseExamples.Quickstart\Client".
Run the "FootlooseExamples.Quickstart.Client.exe".

The binaries of the service are located in "Build\FootlooseExamples.Quickstart\Service".
Run the "FootlooseExamples.Quickstart.Service.Runner.exe".

As soon as both console applications are running you can follow the instructions on
the client's console screen to execute methods on the service.
