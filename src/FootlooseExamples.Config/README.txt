Footloose Configuration Example

1.) Description

This example shows how to confgure and create a FootlooseService instance
with the .NET Configuration, e.g. the App.Config or by Code, and the Footloose fluent configuration API.



2.) How to build?

Use the provided "build-all.bat" in the Examples root directory or open the
"FootlooseExamples.Config.sln" solution file with VS2010 and run
Build -> Build Solution in the VS2010 menu.



3.) How to run?

You'll find the binaries of the examples in "Build\FootlooseExamples.Config\*".
Run the "FootlooseExamples.Config.*.exe".
