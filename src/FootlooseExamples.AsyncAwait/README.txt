Footloose AsyncAwait Example

1.) Description

This quick start will help you to understand on how to use Footloose with the new C# async await keywords. 



2.) How to build?

Use the provided "build-all.bat" in the Examples root directory or open the
"FootlooseExamples.AsyncAwait.sln" solution file with VS2010 and run
Build -> Build Solution in the VS2010 menu.



3.) How to run?

You'll find the binaries of the client in "Build\FootlooseExamples.AsyncAwait\Consumer".
Run the "FootlooseExamples.AsyncAwait.Consumer.exe".

The binaries of the service are located in "Build\FootlooseExamples.AsyncAwait\Service".
Run the "FootlooseExamples.AsyncAwait.Service.exe".

As soon as both console applications are running you can follow the instructions on
the client's console screen to execute methods on the service.
